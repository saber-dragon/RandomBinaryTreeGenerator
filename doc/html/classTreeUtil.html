void TreeUtil::SimplePrint ( std::ostream &  os)
inline

Simple print.

This function simply print all nodes of the tree in the format: {val : cur->val, left : cur->left->val, right : cur->right->val}. Note that if left/right does not exists, a string of "NULL" will be used..
